## 🎨 **Usage Examples**

### **Form Card:**
```css
background: #1e2a3a;
color: #ffffff;
border: 1px solid #2d3f5f;
box-shadow: 0 20px 60px rgba(0, 0, 0, 0.7);
```

### **Input Field:**
```css
background: #2a3544;
color: #ffffff;
border: 2px solid #3d4f6f;
placeholder: #8b9dc3;

/* Focus */
background: #34425a;
border: 2px solid #5b7bb4;
box-shadow: 0 0 0 4px rgba(91, 123, 180, 0.3);
```

### **Submit Button:**
```css
background: linear-gradient(135deg, #4a6fa5 0%, #3d5a8c 100%);
color: #ffffff;
box-shadow: 0 8px 20px rgba(74, 111, 165, 0.5);
```

### **Link:**
```css
color: #6b9bd1;
```
