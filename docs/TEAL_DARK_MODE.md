# 🌊 Teal/Cyan Dark Mode Theme

## Overview
Dark mode has been updated to a beautiful light greenish-blue (teal/cyan) color scheme with white text for better aesthetics and readability.

## 🎯 **Usage Examples**

### **Form Card:**
```css
background: #1a5f5f;
color: #ffffff;
border: 1px solid #2d7a7a;
```

### **Input Field:**
```css
background: #236b6b;
color: #ffffff;
border: 2px solid #3d8f8f;
placeholder: #b0e0e0;

/* Focus */
background: #2d7f7f;
border: 2px solid #4dd4d4;
box-shadow: 0 0 0 4px rgba(77, 212, 212, 0.3);
```

### **Submit Button:**
```css
background: linear-gradient(135deg, #20b2aa 0%, #17a2b8 100%);
color: #ffffff;
box-shadow: 0 8px 20px rgba(32, 178, 170, 0.5);
```

### **Link:**
```css
color: #5dffff;
```
